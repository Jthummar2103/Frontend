# Student Resource Management System

## Project Overview
This is a full-stack web application that allows users to manage learning resources. Users can register, log in, and perform CRUD operations on resources categorized into different types.

---

## Features

- User Registration & Login (JWT Authentication)
- Protected Routes (Only logged-in users can access)
- Add Resource
- View Resources
- Delete Resource
- Category-wise classification
- Responsive UI with clean design

---

## Tech Stack

### Frontend:
- React (Vite)
- React Router DOM
- Axios

### Backend:
- Node.js
- Express.js
- MySQL
- JWT Authentication
